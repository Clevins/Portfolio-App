## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation


1. Clone the repo
   ```sh
    https://github.com/Clevins/Portfolio-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run Vite command
   ```js
    npm run dev
   ```
